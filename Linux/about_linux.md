## ターミナル

入出力の画面を提供するソフトウェア。（画面の担当）<br>
実際にユーザーがコマンドを入力する画面。<br>
ターミナルの中でシェルが動いている。

<br>
<br>

---

## shell

ユーザーと Linux の仲介役。（コマンドを解釈して Linux とやり取りする担当）<br>
シェルは Linux のコマンドを受け取って結果を出力するためのソフトウェア。<br>
bash もシェルの一種。

```text: Linuxのコマンド実行の流れ
→ ユーザーがコマンドを入力
→ シェルが命令を解釈してLinuxに伝える
→ Linuxがコマンド実行
→ シェルが実行結果をユーザーに伝える
```
